# Анализ температурных данных

Приложение Streamlit для анализа исторических температур.

## Структура проекта

```
.
├── app.py                  # Точка входа
├── pages/
│   ├── 1_Main.py           # Главная страница
│   └── 2_Experiments.py    # Эксперименты (sync/async, parallel)
├── utils.py                # Вспомогательные функции и логика анализа
├── data/
│   └── temperature_data.csv
├── requirements.txt
└── README.md
```

## Функциональность

**Главная страница**

* анализ температурных временных рядов по городам
* описательная статистика
* поиск аномалий:

  * по скользящему окну
  * по сезонным отклонениям
* визуализация результатов

**Эксперименты**

1. **Sync vs Async**

   * запрос текущей температуры по нескольким городам
   * сравнение синхронных и асинхронных HTTP-запросов
   * измерение времени и ускорения

2. **Sequential vs Parallel**

   * исторический анализ температур по всем городам
   * сравнение последовательных и параллельных вычислений
   * настройка размера окна и числа потоков
   * подсчёт количества аномалий

## Выводы по экспериментам

* **Async** значительно быстрее синхронного подхода при работе с несколькими городами.
* **Параллельные вычисления** ускорения не дают, даже, наоборот, замедляют вычисления.

## Запуск

```
pip install -r requirements.txt
streamlit run app.py
```
